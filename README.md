# Front Template

This is a front template project.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Stacks

- Nextjs
- Styled components
- Typescript
- Storybook
- React testing Library
- Jest
- Eslint
- Prettier
